# MinistryLion - Sistema de Gestión de Conferencias (MLSGC)

Sistema de control y gestión de conferencias religiosas.

## 🚀 Despliegue

Este proyecto está desplegado en GitHub Pages: https://depregpsac-lgtm.github.io/MLSGC/

## 📦 Base de Datos

La base de datos está en Supabase. Para configurar:

1. Crear proyecto en Supabase
2. Ejecutar `database.sql` en el SQL Editor
3. Copiar URL y API Key
4. Crear `config.js` con las credenciales

## 📁 Estructura

- `index.html` - Página de login
- `control-conferencias.html` - Dashboard principal
- `style.css` - Estilos
- `config.js` - Configuración de Supabase (no subir a GitHub)
- `app.js` - Lógica de la aplicación
- `database.sql` - Script de base de datos
